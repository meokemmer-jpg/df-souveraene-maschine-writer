## Konzept

K_0-AI als Souveraenitaets-Werkzeug. Kapitalerhaltung + Familie + Freiheit als
mathematisch begrueendete Invarianten. Die "souveraene Maschine" ist nicht
super-intelligent, sondern constraint-treu.

Kernthese: Souveraenitaet entsteht nicht durch maximale AI-Macht, sondern durch
geschickte K_0/Q_0/I_min-Constraints. Die Maschine die nicht alles tut, was sie
koennte - das ist Souveraenitaet.
